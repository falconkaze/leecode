# 算法

## 排序算法

### 比较排序算法

| 算法     | 最坏情况运行时间 | 平均情况/期望运行时间 |
| :---:    | :---             | :---                  |
| 插入排序 | O(n^2)           | O(n&2)                |
| 归并排序 | O(nlgn)          | O(nlgn)               |
| 堆排序   | O(nlgn)          | --                    |
| 快速排序 | O(n^2)           | O(nlgn)               |

### 其他排序算法

| 算法       | 最坏情况运行时间 | 平均情况/期望运行时间 |
| :---:      | :---             | :---                  |
| 计数排序   | O(k+n)           | O(k+n)                |
| 归并排序   | O(d(n+k))        | O(d(n+k))             |
| 桶排序排序 | O(n^2)           | O(n) （平均情况）     |
